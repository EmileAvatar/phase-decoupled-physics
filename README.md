## Inertia (Reframed)

Inertia and gravity arise from the same mechanism:

- How strongly matter-waves are locked to spacetime-waves

Strong lock:
- Normal gravity
- Normal inertia

Weakened lock:
- Reduced gravity
- Reduced inertia

## Phase-Decoupling

Phase-decoupling means:

Reducing how strongly matter’s internal waves synchronize with spacetime waves.

This does NOT mean:

- Creating anti-gravity
- Creating negative mass
- Shielding gravity

It DOES mean:

- Altering coupling strength

Analogy:
Noise-canceling headphones don’t block sound.
They manipulate phase relationships.
